# 通所介護モニタリング作成アプリアイコン

## 入っているファイル
- apple-touch-icon.png
- favicon-16x16.png
- favicon-32x32.png
- favicon-48x48.png
- favicon.ico
- icon-152.png
- icon-167.png
- icon-180.png
- icon-192.png
- icon-512.png
- site.webmanifest
- html-head-snippet.txt

## 使い方
1. ZIPを展開します。
2. 既存の同名ファイルを置き換えます。
3. index.html の `<head>` 内が未設定の場合は、html-head-snippet.txt の内容を貼り付けます。
4. GitHub Pages等で公開している場合は、反映後にブラウザキャッシュを更新してください。

※ index.html / install.html は既存のものをそのまま使う前提です。
